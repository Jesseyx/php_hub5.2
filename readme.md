## 项目概述

* 产品名称：PHPHub5.2
* 项目代码：PHPHub5.2
* 该项目为学习 [Laravel](https://laravel.com/) 时参考 PHPHub5 [源码](https://github.com/summerblue/phphub5) 所写, PHPHub5 使用 Laravel 5.1 版本，该项目修改了部分代码使其兼容 Laravel 5.2
* 具体信息请参见 [PHPHub5](https://github.com/summerblue/phphub5)
